# Api-Django

Iniciar
-------------

Descargar repositorio
`git clone git@github.com:ginop90/test.git`

Cambiar de directorio
`cd 1-api-django/backend`

Otorgar permiso de ejecución
`chmod +x run.sh`

Correr script
`./run.sh`

Ingresar por
http://localhost/

Finalizar
-------------

Cambiar de directorio
`cd 1-api-django/backend`

Otorgar permiso de ejecución
`chmod +x delete.sh`

Correr script
`./delete.sh`
